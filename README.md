# NestJS Backend Tester

A visual API client to exercise NestJS backends — built with **Next.js + shadcn/ui**.
Send requests, inspect responses, test auth flows and payloads without leaving the browser.

Companion tool for [nest-backend-template](https://github.com/guillerg01/nest-backend-template).

## Stack

- **Next.js** (App Router) + **TypeScript**
- **shadcn/ui** + **Tailwind CSS**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- Endpoint explorer with request builder
- Auth flow testing (JWT / refresh tokens)
- Response inspection and status feedback
- Clean, themeable UI (light / dark)
